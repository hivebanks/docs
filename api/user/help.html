HiveBanks用户API说明文档

2018.5.14 V1.0 by ArthurYuan

概要说明

            API访问地址
              测试服：http://doc.fnying.com/api/user/help.html
              正式服：待定

            返回数据格式
              json,支持jsonp(callback)调用
              errcode 错误码
              errmsg 错误信息

            正常返回
              errcode = 0
              errmsg = ''
              其他需要返回的json格式数据

            异常返回
              常见错误码：
              -1  系统繁忙
              109 非正确的认证信息
              114 TOKEN过期
              131 参数错误

发送手机短信验证码

http://testapi.fnying.com/user/sms_send.php 

            GET参数
            cellphone                 手机号码
            country_code              国家代码
            bind_type                 申请验证的类型（1：注册 2：登录 3：忘记密码 4：绑定 5：交易）
            返回
              cfm_code                短信验证码

            说明
              手机通过该接口获取的验证码有效期为5分钟

              text_hash       文本内容HASH
              pass_word_hash  密码HASH

            返回
              errcode = 0     请求成功

            说明
              绑定手机，邮箱，姓名，身份证，第三方账号等
