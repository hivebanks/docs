HiveBanks用户API说明文档

2018.5.14 V1.0 by ArthurYuan

概要说明

            API访问地址
              测试服：http://testapi.fnying.com/la
              正式服：待定

            返回数据格式
              json,支持jsonp(callback)调用
              errcode 错误码
              errmsg 错误信息

            正常返回
              errcode = 0
              errmsg = ''
              其他需要返回的json格式数据

            异常返回
              常见错误码：
              -1  系统繁忙
              110 系统异常
              114 TOKEN过期
              120 参数错误

获取ba列表

http://testapi.fnying.com/la/la_admin/manage/ba_list.php 

            GET参数
            token            用户token
            返回
            rows            信息数组
              ba_id        baID
              ba_nm         代理商编号（内部唯一）
              ba_account    代理商表示账号（内部唯一）
              base_amount   基准资产余额
              lock_amount   锁定余额
              ba_type       代理商类型
              ba_level      代理商等级
              security_level安全等级
              utime         更新时间
              ctime         创建时间
