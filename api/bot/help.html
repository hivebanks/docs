概要说明

            API访问地址
              测试服：http://doc.fnying.com/api/bot/help.html
              正式服：待定

            返回数据格式
              json,支持jsonp(callback)调用
              errcode 错误码
              errmsg 错误信息

            正常返回
              errcode = 0
              errmsg = ''
              其他需要返回的json格式数据

            异常返回
              常见错误码：
              -1  系统繁忙
              109 非正确的认证信息
              114 TOKEN过期
              131 参数错误
